### [Quality control and filtering](https://github.com/talimass/Cayman-translocation/tree/main/RNAseq_analysis/QC_Filtering) 
The initial QC analysis of raw data was performed using FastQC v0.12.1: [fastqc.sh](https://github.com/talimass/Cayman-translocation/blob/main/RNAseq_analysis/QC_Filtering/fastqc.sh). MultiQC v1.0 was used to create a report: [multiqc.sh](https://github.com/talimass/Cayman-translocation/blob/main/RNAseq_analysis/QC_Filtering/multiqc.sh). After inspecting the results, raw reads were trimmed with Trimmomatic v0.39 using a custom adaptor [file](https://github.com/talimass/Cayman-translocation/blob/main/RNAseq_analysis/QC_Filtering/Sequencing_adaptors.fasta): [trimmomatic.multi.sh](https://github.com/talimass/Cayman-translocation/blob/main/RNAseq_analysis/QC_Filtering/trimmomatic.multi.sh). The resulting read quality was accessed again with FastQC and summarized with MultiQC. The performance of rRNA depletion was tested with SortMeRNA v.4.3.6 tool with [sortmerna.multi.sh](https://github.com/talimass/Cayman-translocation/blob/main/RNAseq_analysis/QC_Filtering/sortmerna.multi.sh) and summarized with MultiQC.
